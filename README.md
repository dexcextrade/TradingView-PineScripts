# 📊 Trading Strategies for Crypto Futures

Стратегии под личную настройку для фьючерсной торговли на криптотокенах.

Проект включает набор торговых логик и индикаторов, которые можно адаптировать под свой стиль торговли (скальпинг / интрадей / свинг).

---

## ⚙️ Особенности

- 📈 Подходит для крипто-фьючерсов
- 🧠 Комбинация индикаторов (RSI, MACD, Supertrend, MA)
- ⚖️ Взвешенная система сигналов
- 🔧 Гибкая настройка параметров
- 📉 Подходит для backtest в TradingView

Данный проект не является финансовой рекомендацией.

Все стратегии требуют:

- оптимизации под рынок
- управления рисками

---

## 📌 Цель проекта

Создание гибкой системы торговых стратегий, которые можно:
- адаптировать под рынок
- комбинировать между собой
- использовать для анализа и автоматизации решений
